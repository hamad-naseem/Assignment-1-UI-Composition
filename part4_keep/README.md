### Getting Started

1. Clone this repository:

   ```bash
    git clone https://github.com/hamad-naseem/Assignment-1-UI-Composition.git
   cd part4_keep
   ```

2. Get dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app:

   ```bash
   flutter run
   ```
